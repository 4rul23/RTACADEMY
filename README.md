## Apa itu RT-ACADEMY?

RT-ACADEMY adalah Sebuah Aplikasi Website Yang Dibuat Untuk Melatih para siswa-siswa SMK/SMA indonesia Singkatnya ini adalah Website untuk melatih keterampilan siswa-siswi SMK /SMA indonesia

TLDR
